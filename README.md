# StarFighters Arcade

Redesign of starfightersarcade.com — Mesa, AZ retro 80s arcade.

**Stack:** Next.js 15 (App Router) · Tailwind CSS · TypeScript
**Hosting:** Coolify (Contabo VPS 2) at `starfighters.advancedmarketing.co`

## Dev

```bash
npm install
npm run dev
```

## Production

Dockerfile-based build. Coolify builds and runs the standalone Next.js output on port 3000.
